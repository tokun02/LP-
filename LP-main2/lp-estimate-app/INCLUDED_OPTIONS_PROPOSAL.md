#### 📝 **KPI・コンセプト作成とは？**

Web制作における**戦略的プランニングサービス**です。具体的には以下の作業を含みます：

**1. KPI（重要業績評価指標）設定**
- サイトの成功指標を定義
  - 例：問い合わせ数を30%増加、資料請求数を月50件達成、CV（コンバージョン）率を5%向上など
- 計測方法の設計（GA4設定、目標設定など）
- 成果評価の基準を明確化

**2. コンセプトワーク**
- サイトの方向性・コンセプトを明確化
  - ターゲットユーザーの定義
  - ブランドメッセージの整理
  - 競合との差別化ポイントの抽出
  - サイトの目的・ゴール設定
- コンテンツ戦略の基本方針策定

**3. ワークショップ実施（1回）**
- クライアントとの対話形式のミーティング
- ヒアリング・ブレインストーミング・合意形成
- 通常2〜3時間程度のセッション

**含まれる成果物:**
- KPI設定シート
- コンセプトメモ/資料
- ワークショップ議事録
- 次のステップ（サイト設計など）への方向性

**なぜ高価格？**
- 戦略的な思考作業であり、制作物の方向性を決める重要な工程
- クライアントとのワークショップ実施により工数が大きい
- サイト制作の成功確率を大きく左右する高付加価値サービス

**AI普及の影響（2024-2025年現在）**

## 🌐 多言語対応の料金設定について

### 現在の設定

**多言語対応: ¥40,000/言語**
- 追加言語1言語あたりの料金
- 翻訳と多言語導線設計を含む
- 計算方式: 選択した言語数に応じて料金が加算される

**例:**
- 1言語追加: ¥40,000
- 2言語追加: ¥80,000
- 3言語追加: ¥120,000

### 市場相場の調査結果（2024-2025年）

**1言語追加あたりの相場:**
- **基本範囲**: ¥30,000〜¥60,000/言語（翻訳+基本的な導線設計）
- **総合的な対応**: ¥100,000〜¥300,000/言語（翻訳+デザイン調整+システム実装+SEO対策を含む）

**現在の設定（¥40,000/言語）の評価:**
- ✅ **市場相場の範囲内**（¥30,000〜¥60,000の範囲）
- ✅ 翻訳と基本的な導線設計を想定した適正価格
- ⚠️ デザイン調整やシステム実装の詳細度によっては追加費用が必要な場合もある

**価格に含まれる作業（現在の設定）:**
- テキスト翻訳（ネイティブチェック含む可能性）
- 多言語導線設計（言語切替機能の基本実装）
- 基本的なSEO設定（言語別メタタグなど）

**追加で別途費用が発生する可能性がある作業:**
- デザイン・レイアウトの詳細な調整（文字数や文字方向の違いへの対応）
- 高度な多言語CMS実装
- 言語別の詳細なSEO対策
- 運用・保守（各言語ごとの更新対応）

### 結論

**現在の設定（¥40,000/言語）は市場相場の範囲内で適正**

ただし、以下の点に注意：
- 基本料金には「翻訳+基本的な導線設計」のみが含まれる
- デザイン調整や高度な実装が必要な場合は、別途見積もりが必要
- 言語数が増えるごとに料金が累積する（1言語¥40,000、2言語¥80,000...）

### ページ基本料金について

複数ページが基本料金に含まれているパッケージ（採用サイト、ECサイト、オウンドメディアなど）については、**ページ基本料金**として明記。

- **ページ基本料金**: パッケージに含まれるページ数分の制作費用（レスポンシブ最適化含む）
- **追加ページ**: 基本料金に含まれるページ数を超える追加ページ分の費用
  - HPパッケージ: 1ページ追加ごとに¥15,000

## 📈 実装方法

### システム変更点

1. **`BasePackage` 型の拡張**
   ```typescript
   type BasePackage = {
     // ... 既存フィールド
     includedOptions: {
       new: OptionCode[];      // 新規作成時に含まれるオプション（レスポンシブのみ）
       renewal: OptionCode[];  // リニューアル時に含まれるオプション（レスポンシブのみ）
     };
     // includedPagesは「ページ基本料金に含まれるページ数」として明記
   };
   ```

2. **計算ロジックの変更**
   - `includedOptions` に含まれるオプション（レスポンシブ最適化、アニメーション（基本）、サイト設計（基本）、デザイン制作（基本））は、追加料金として計算しない
   - `includedPages` は「ページ基本料金」として明記
   - 見積書には「基本料金に含まれるオプション」として明示的に表示（0円表示）
   - 複数ページが含まれるパッケージは「ページ基本料金（Xページ分）」として表示

3. **見積書表示例**
   ```
   基本料金（HP・新規作成）
   ```
